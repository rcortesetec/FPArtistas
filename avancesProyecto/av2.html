Avance de proyecto 2 :

Subir a Blackboard de forma individual lo siguiente:

  • Carpeta Zip: Agregar a una carpeta el boceto realizado para la primera entrega escaneado y aparte el documento de Processing con tu monstruo ya computarizado. Todo esto se deberá de estar todo junto dentro de una carpeta comprimida.
  • Características: El monstruo computarizado deberá de ser igual al del boceto. Al igual que la primera entrega, debe de ser un dibujo en blanco y negro.
  • Consejos: Intenta implementar el uso de ciclos, ciclos anidados, vértices para optimizar el código de dibujo de tu monstruo. Mientras tu código este más simplificado, será más fácil de realizar una modificación posteriormente.

Importante: En caso de querer hacer una modificación acerca de las características de tu monstruo, hablarlo directamente con el profesor y llegar a un acuerdo formal.
